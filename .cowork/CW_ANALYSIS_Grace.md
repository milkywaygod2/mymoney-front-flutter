### 도메인 개념

**손익 계정 분류 체계**:
- 영업수익: 광고 / 비즈니스플랫폼 / IT플랫폼 / 콘텐츠서비스 / 라인 및 기타 플랫폼
- 영업비용: 플랫폼 개발/운영 / 대행/파트너 / 인프라비 / 마케팅비 / 라인 및 기타
- 영업이익 + 영업이익률(%)
- 영업외 수익: 기타수익 / 금융수익
- 영업외 비용: 기타비용 / 금융원가
- 지분법손익 (관계기업 지분법 평가 결과)
- 법인세비용
- 중단사업손익
- 당기순이익 + 당기순이익률(%)

**재무상태표 핵심 계정**:
- 자산: 유동(현금및현금성자산 / 단기금융상품 / 상각후원가측정단기금융상품 / 당기손익공정가치측정금융자산 / 매출채권및기타채권 / 재고자산 / 단기투자자산 / 기타유동자산) + 비유동(유형자산 / 무형자산 / 투자자산 / 비유동이연법인세자산 / 기타비유동자산)
- 부채: 유동(매입채무및기타채무 / 단기차입금 / 미지급법인세 / 기타충당부채 / 기타유동부채) + 비유동(장기채무 / 장기차입금 / 퇴직급여부채 / 비유동기타충당부채 / 비유동이연법인세부채 / 기타비유동부채)
- 자본: 지배기업귀속자본(자본금 / 자본잉여금 / 기타자본구성요소 / **기타포괄손익누계액** / 이익잉여금) + 비지배지분

**플랫폼별 손익**: 네이버 / 라인 / 스노우 / 전체 (사업부문 세그먼트)

**요약 재무제표 (P/L + B/S 핵심 항목)**:
- P/L 9항목: 영업수익 / 영업비용 / 영업이익 / 영업이익률 / 영업외수익 / 영업외비용 / 법인세 / 중단사업손익 / 당기순이익 / 당기순이익률

**부문별 영업비용 집계**: 플랫폼 개발/운영 / 대행/파트너 / 인프라비 / 마케팅비 — 합계+네이버+라인+스노우 4열

**연결범위 변동 추적** (월별 법인 수):
- 연결 법인 수: 88개(Oct-17) → 98개(Jun-18)
- 네이버 계열 / LINE 계열 / CAMP 계열 / SNOW 계열 + 지분법 법인 수
- 2018.06 기준: 연결 98개사 + 지분법 65개사 = 163개사

**결산 코멘트** (비정형 텍스트 행):
- 영업수익 변동 원인: "LINE Mobile 지분율 하락(100%→49%)으로 인한 처분이익 -1,127억"
- 부채 변동 원인: "단기차입금: SNOW China +115억(외부투자금 차입금 분류)"
