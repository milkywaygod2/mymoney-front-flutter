### GAP 상세

- **OCI 누계액 계정**: 기타포괄손익누계액이 자본의 독립 항목으로 존재 — CW_ARCH 자본 분류에서 "기타자본"에 통합됨
- **비지배지분**: 연결재무제표 자본의 별도 구분 항목
- **지분법손익**: 관계기업 투자 지분법 평가 결과 P/L 반영
- **중단사업손익**: IFRS 5 — 매각 결정 사업부문 별도 표시
- **이연법인세**: 회계이익-과세소득 일시적 차이
- **사업부문 세그먼트 집계**: DimensionType SEGMENT 미정의
- **증감률 계산 레이어**: ReportQueryService에 없음
- **결산 코멘트**: FiscalPeriods 테이블에 note/comment 컬럼 없음

### GAP 상세 (시트 11 신규)

1. **Rolling 12M 이동합산**: ROA/ROE 분자(순이익)는 단월이 아닌 직전 12개월 합산 — 단순 `periodId` 조회로는 불가, `findByDateRange()`로 12개월 조회 후 합산 필요
2. **평균 산출 (T+T-12)/2**: 분모(총자산/자기자본/매출채권)는 당월말+12개월전말의 평균
3. **미구현 예정 비율 16종**: ROIC/EBITDA마진/유동자산증가율/유형자산증가율/자기자본증가율/유동부채비율/비유동부채비율/순부채비율/당좌비율/금융비부담률/재고자산회전율/순운전자본회전율/유형자산회전율/매입채무회전율/(PER?/EPS? 별도 슬롯)
4. **외부 데이터 연동**: PER/EPS 계산에 주가, 주식수 필요 — LegalParameter 또는 별도 ExternalData 엔티티

## 결론

**CW_ARCHITECTURE.md는 복식부기 핵심, 외환, 결산 5단계, 세무조정 영역을 잘 커버한다.**

**즉시 보완이 필요한 2건 (P1)**:

- M-02 재무비율 엔진: 구현 13종 + 미구현 16종 = 29종. Rolling 12M + 평균 산출 패턴 필수.

**단기 보완 7건 (P2)**:
- 기간 비교(4종 동시), 비지배지분, 충당부채(L4 5세분류), 이익잉여금(5종), 중단사업(이원구조), FiscalPeriod 메모, 매각예정자산/부채
