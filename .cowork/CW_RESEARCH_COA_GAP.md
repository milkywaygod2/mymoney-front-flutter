## 현재 §12.1 계정과목 현황

```
자산
  유동: 현금성 / 단기금융 / 매출채권 / 선급 / 재고
  비유동: 투자 / 유형 / 무형 / 기타
부채
  유동: 미지급 / 단기차입 / 예수 / 선수
  비유동: 장기차입 / 기타
자본
  자본금 / 이익잉여금 / 기타자본
수익
  영업(급여/사업/매출) / 금융(이자/배당/외환차익) / 투자(처분이익) / 기타
비용
  생활(식비/교통/통신/주거/의료/교육) / 영업(급여/복리후생/접대/여비)
  금융(이자/외환차손) / 감가상각 / 세금 / 기타
```

**문제**: L2(대분류)까지만 정의. L3~L4 세부 경로 없음. 가계부 생활비 카테고리 중심이라 기업 회계 필수 계정이 대폭 누락.

## 특별 분석: 재고자산/매출원가/제조원가

### 재고자산 세부 (현재 완전 누락)

```
ASSET.CURRENT.INVENTORY
  ├ MERCHANDISE          -- 상품 (매입하여 그대로 판매)
  ├ FINISHED_GOODS       -- 제품 (자체 제조 완성품)
  ├ WORK_IN_PROGRESS     -- 재공품 (제조 중)
  ├ RAW_MATERIALS        -- 원재료
  ├ SUPPLIES             -- 저장품/소모품
  └ GOODS_IN_TRANSIT     -- 미착품 (운송 중)
```

**가계부 적용**: 개인 사업자(스마트스토어, 쿠팡 셀러)의 상품 재고 관리 필요. P2에서 `MERCHANDISE`(상품)만 우선 추가, 나머지는 P3(사업 확장).

### 매출원가 구성 (현재 완전 누락)

```
EXPENSE.COGS                     -- 매출원가
  ├ BEGINNING_INVENTORY          -- 기초재고
  ├ PURCHASES                    -- 당기매입
  │   ├ PURCHASE_COST            -- 매입원가
  │   ├ PURCHASE_FREIGHT         -- 매입운반비
  │   └ PURCHASE_RETURNS         -- 매입할인/환출 (차감)
  ├ ENDING_INVENTORY             -- 기말재고 (차감)
  └ MANUFACTURING_COST           -- 당기제조원가 (제조업만)
```

**공식**: `매출원가 = 기초재고 + 당기매입 - 기말재고`

**가계부 적용**: 사업자가 아닌 개인 가계부에서는 불필요. 사업자 확장 시 P3에서 추가.

### 제조원가 계정 (현재 완전 누락)

```
EXPENSE.MANUFACTURING            -- 제조원가
  ├ MATERIALS                    -- 원재료비 (직접/간접)
  ├ LABOR                        -- 노무비 (직접/간접)
  │   ├ DIRECT_LABOR             -- 직접노무비
  │   └ INDIRECT_LABOR           -- 간접노무비
  └ OVERHEAD                     -- 제조경비
      ├ DEPRECIATION             -- 감가상각비 (제조설비)
      ├ UTILITIES                -- 수도광열비
      ├ MAINTENANCE              -- 수선비
      └ OTHER_OVERHEAD           -- 기타 제조경비
```

**가계부 적용**: 개인 가계부에서 제조업을 운영하는 경우는 극히 드묾. **P4(장기 예약)**.
